# Web
### 웹사이트
- 웹페이지(문서)들의 모음을 링크로 연결한 것
- 글을 HTML 형식으로 작성

### 웹사이트의 구성요소
- HTML = 건물구조
- CSS = 인테리어(표현)
- Javascript = 엘레베이터(동작)
