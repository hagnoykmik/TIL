# Today I Learnd

## 1. Why git and github?

### git?

분산 버전 관리 시스템

- 버전관리
    - 백업
    - 복구
    - 협업

- 분산 저장 원리

모든 사용자가 똑같이 완전한 버전을 가지고 있음

### github?

git을 기반으로 한 원격저장소로서 포트폴리오로서 증명가능 


### CLI
: Command Line Interface
git은 CLI형식으로 소통


### 기본명령어
``` 
mkdir 폴더만들어줘
ls 목록 보여줘
ls -a 숨김 폴더까지 다 보여줘
touch 파일 만들어줘
rm 파일 없애줘
rm -r recursive 폴더 지워줘
cd 이동해줘
cd .. 상위 폴더로 이동해줘
```
